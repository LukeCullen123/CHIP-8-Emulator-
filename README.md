# CHIP-8 Emulator

This is a CHIP-8 emulator written in C.

## Features
- Emulates the CHIP-8 virtual machine.
- Supports basic CHIP-8 instructions.

## Requirements
- CMake 3.10 or higher
- SDL3 library

## Build Instructions
1. Clone the repository:


2. Create a build directory and run CMake:
   ```bash
   mkdir build
   cd build
   cmake ..
   ```

3. Build the project:
   ```bash
   cmake --build .
   ```

4. Run the emulator:
   ```bash
   ./chip-8
   ```
